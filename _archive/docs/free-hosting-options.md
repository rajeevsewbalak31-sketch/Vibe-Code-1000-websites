**Website**: https://vercel.com  
**Type**: Static hosting, Serverless Functions  
**Free Tier**: Unlimited static sites, 100GB bandwidth/month

#### Pros:
✅ Extremely fast (global CDN)  
✅ Seamless GitHub integration  
✅ Automatic deployments  
✅ Serverless functions included  
✅ Great for Next.js  
✅ Custom domains free  
✅ SSL certificates automatic  

#### Cons:
❌ Limited free tier for serverless  
❌ Can be pricey for heavy usage  
❌ Bandwidth limits  

#### Best For:
- React apps
- Next.js applications
- Static sites
- Personal portfolios

#### Cost Breakdown:
```
Free Tier:
- Unlimited deployments
- 100GB bandwidth/month
- Unlimited static sites
- Free domain (yourname.vercel.app)

Paid (if needed):
- Pro: $20/month (Analytics, priority support)
- Enterprise: Custom pricing
```

#### Hidden Costs for 1000 Sites:
**$0/month** (stays within free tier)

## Free vs Paid Decision Tree

```
Do you need a database?
├─ NO → Use Vercel, Netlify, or GitHub Pages (FREE)
└─ YES → Use Railway ($20/mo) or Firebase ($20/mo)

Do you need serverless functions?
├─ NO → Use GitHub Pages (FREE)
├─ YES → Use Vercel or Netlify (FREE)
└─ BACKEND API → Use Railway ($20/mo)

Do you need real-time features?
├─ NO → Use any platform (FREE-$20/mo)
└─ YES → Use Firebase ($20/mo)

Do you need custom domain?
├─ NO → Use any platform (FREE)
└─ YES → Use GitHub Pages or Vercel (FREE)
```
